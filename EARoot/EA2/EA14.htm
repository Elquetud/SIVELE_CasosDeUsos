Details:
El apartado de ventas tendrá como función ser el receptor de ventas realizadas para luego de ser verificadas y autorizadas llevar la orden al apartado de entrega-producto.<br/>
Steps:
1. El servidor tendra el control de compras y la organización de los productos y su distribución.<br/>
2. El servidor enviara los datos de envíos de las compras mediante una api al correo electrónico.<br/>
